# Atlas of Weyl Chambers for E₈ — Complete Scan of 120 Positive Roots

## 📖 About
Full algorithmic scan of all 120 positive roots of E₈ as seeds for simple root discovery. Found exactly 8 unique Weyl chambers, all sharing the same set of fundamental dimensions.

## 🔬 Key Results
• **8 unique Weyl chambers** discovered
• **1 unique dimension set** — all chambers have identical fundamental dimensions
• **Cartan invariants:** trace = 16, determinant = 1 (E₈ invariants)
• **0 prime values** among fundamental dimensions
• **Bourbaki match:** 5/8 dimensions per chamber
